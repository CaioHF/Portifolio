Projeto Lista de Tarefas

Um projeto de lista de tarefas interativa com barra de progresso, onde é possível adicionar, editar e excluir tarefas, acompanhando visualmente o andamento do dia.

Saiba mais...

Este projeto foi desenvolvido utilizando HTML, CSS e JavaScript com o objetivo de criar uma lista de tarefas dinâmica e intuitiva. O usuário pode adicionar novas atividades, editar ou remover tarefas concluídas, enquanto uma barra de progresso exibe em tempo real o percentual de conclusão. O design foi pensado para ser simples, funcional e responsivo, ajudando no gerenciamento das tarefas diárias e na organização pessoal de forma prática e visualmente agradável.
